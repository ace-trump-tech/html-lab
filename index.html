📜 仙府简介

摆烂仙君 闭关修炼，已炼成三十件仙器（在线工具），涵盖编解码、格式化、生成器、图片处理、文本处理诸般法门。所有仙法皆以 HTML/CSS/JS 为根基，不借外力，道友的数据绝不出仙府。

🚀 炼器总览

🪄 镇阁仙器

  • Base64 编码/解码（UTF-8 / 中文 / Emoji）
  • 二维码生成器（QR Code · 下载 PNG）
  • Markdown 在线预览（实时渲染 · 导出 HTML）
  • 文本差异对比（逐行 Diff · 增删高亮）
  • 字数 & 字节统计器（UTF-8 · 阅读时间）

🖼️ 图片处理

  • 图片压缩（质量/缩放可调）
  • 图片格式转换（PNG / JPEG / WebP）
  • 图片 Base64 转换（图片 ⇄ Base64）

🔤 编码/解码

  • URL 编码/解码（encodeURIComponent / encodeURI）
  • HTML 实体编解码（&lt; ⇄ <）
  • Unicode 编解码（\\uXXXX ⇄ 原文）
  • 摩斯电码（文本 ⇄ Morse Code）
  • JWT 解码器（Header / Payload / Claims）

🎲 生成器

  • 随机密码生成器（Crypto API · 强度显示）
  • UUID 生成器（v4 · 批量生成）
  • Lorem Ipsum 生成器（拉丁/中文假文）
  • 哈希生成器（MD5 / SHA-1 / SHA-256 / SHA-512）

📐 格式化/校验

  • JSON 格式化 & 校验（语法高亮 · 压缩 · 转义）
  • HTML 格式化（美化 · 压缩）
  • CSS 格式化（美化 · 压缩）
  • JavaScript 格式化（美化 · 压缩）
  • SQL 格式化（关键字大写 · 换行缩进）

📝 文本处理

  • 正则表达式测试器（实时高亮 · 捕获分组）
  • 大小写转换（UPPER / lower / Title / 驼峰等）
  • 文本去重（逐行去重 · 可选空行）
  • 文本排序（字母/数字/随机）

🧰 其他工具

  • Unix 时间戳转换（实时 · 秒/毫秒）
  • 颜色工具（HEX / RGB / HSL 互转）
  • 进制转换（2 / 8 / 10 / 16）
  • 单位换算（长度/重量/温度/数据）
